# Plataforma de Servicios Profesionales - Gran Logia Argentina

## Descripción del Proyecto

Plataforma digital que conecta a miembros de la Gran Logia de Argentina que ofrecen servicios profesionales (plomería, gasista, abogados, médicos, contadores, etc.) con otros hermanos que necesitan esos servicios.

**Modelo**: Servicio comunitario ad honorem sin fines de lucro

## Modelo de Financiamiento

### Sin Comisiones
- **La plataforma NO cobra comisiones** sobre las transacciones
- Solo se aplican las comisiones inevitables de Mercado Pago (3.99% + IVA)
- Los profesionales reciben el pago completo menos la comisión del procesador

### Costos Reales
- **Desarrollo**: $0 (ad honorem)
- **Año 1**: $300-350 USD (solo infraestructura)
- **Años siguientes**: $1,500-3,600 USD/año (según escala)

### Ahorro para la GL
Comparado con contratar desarrollo externo: **$127,000-221,000 USD** ahorrados en 5 años

## Stack Tecnológico

### Frontend
- Next.js 15 (App Router)
- TypeScript (strict mode)
- Tailwind CSS v4
- shadcn/ui (componentes)
- Zustand + TanStack Query

### Backend
- Next.js API Routes / Hono
- Drizzle ORM
- PostgreSQL (Neon)
- Redis (Upstash)
- Better-auth

### Mobile
- React Native
- Expo

### Infraestructura
- Vercel (Frontend)
- Cloudflare (CDN, R2 Storage)
- Neon (PostgreSQL)
- Upstash (Redis)

### Servicios Externos
- Mercado Pago (Pagos)
- Resend (Email)
- Firebase FCM (Push Notifications)
- Sentry (Monitoring)
